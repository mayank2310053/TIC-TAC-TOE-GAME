Tic Tac Toe Game

A fun and interactive Tic Tac Toe game built using HTML, CSS, and JavaScript. This project allows two players to compete on a 3×3 grid, featuring real-time game updates, winner detection, draw conditions, and a clean user interface. The project demonstrates core frontend development concepts and JavaScript game logic implementation.

Features

- Two-player gameplay
- Real-time winner detection
- Draw game detection
- Restart/New Game functionality
- Responsive and user-friendly interface
- Interactive game board with dynamic updates
- Clean and organized code structure

Technologies Used

- HTML5
- CSS3
- JavaScript 

Purpose

This project was developed to strengthen frontend development skills and gain practical experience in implementing game logic using JavaScript.

Learning Outcomes

- DOM Manipulation
- Event Handling
- Game Logic Development
- Functions and Arrays
- Conditional Statements
- Responsive Web Design

Future Enhancements

- Single-player mode with AI
- Scoreboard system
- Sound effects and animations
